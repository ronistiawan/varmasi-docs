# Why Varmasi?
 
Berikut adalah keunggulan aplikasi Varmasi

## Mudah dan Nyaman 
Aplikasi yang **simple**, **mudah** dan **nyaman** digunakan.
Dilengkapi dengan dark theme sehingga mata tidak sakit kalaupun digunakan dalam waktu yang lama.
Fitur Penjualan, Pencatatan stok, hingga input faktur pembelian barang sangat mudah digunakan.

## Tidak Membutuhkan Interet

Bisa berfungsi 100% tanpa internet. Internet hanya diperlukan jika anda membutuhkan akses jarak jauh melalui internet.

## Data Aman

Data Apotek anda tersimpan di komputer anda sendiri, bukan di server Varmasi. Berbeda dengan kebanyakan aplikasi apotek, 
yang menyimpan data di server mereka.

## Berbasis Web / Remote Access (Internet Required)

Karena berbasis web, Anda dapat mengakses aplikasi anda dengan menggunakan tunnel.

## Harga Muraaaaahhh

Berbeda dengan aplikasi lainnya, Varmasi hanya dibayar **sekali seumur hidup**, selama storage anda tidak rusak.
Betul, **Tidak ada biaya bulanan**.
